# Генеалогическое древо — веб-приложение для семейных деревьев

Клиентское веб-приложение для работы с генеалогическими проектами: визуализация древа, личные дела, медиа, импорт/экспорт GEDCOM и сохранение проектов в формате `.drevo`.

## Возможности

- Создание пустого проекта, открытие `.drevo`, импорт GEDCOM
- Интерактивное древо с центром на персоне или семье
- Настройки поколений, боковых ветвей, размера карточек, темы (`clean` / `forest`)
- Поиск по ФИО, годам и местам
- Личное дело с биографией, связями и медиа
- Режим редактирования: персоны, связи, файлы
- Экспорт листа в PNG/JPEG/PDF и проекта в GEDCOM
- Ручное расположение карточек (режим редактирования)

## Установка

```bash
npm install
npm run dev
```

Приложение откроется по адресу `http://localhost:5173`.

## Сборка

```bash
npm run build
npm run preview
```

## Тесты

```bash
npm test
```

## Формат проекта `.drevo`

ZIP-архив:

- `project.json` — данные проекта (персоны, семьи, медиа-метаданные, настройки)
- `media/` — файлы фото, видео, аудио, документов

## Ограничения браузера

- Данные хранятся локально (IndexedDB + File System Access API где доступен)
- В браузерах без File System Access API используется загрузка/скачивание файлов
- Большие медиа-файлы увеличивают размер проекта

## Управление на древе

| Действие | Клавиши / мышь |
|----------|----------------|
| Перемещение | Drag любой кнопкой, стрелки, WASD |
| Масштаб | Ctrl + колесо, Ctrl +/-, кнопки |
| Прокрутка | Колесо мыши |
| Горизонтальная прокрутка | Shift + колесо |
